# E-Commerce Full Stack Application

## Frontend

Modern and responsive **React** frontend for the E-Commerce platform.

### Project Overview
This is the **frontend** part of a complete full-stack e-commerce application. It consumes the REST API from the [E-Commerce Backend](https://github.com/lensen-degife/ecommerce-backend) repository.

### ✨ Features
- Product browsing and search
- Product details view
- Add / Update products
- Shopping Cart (with modal)
- Responsive UI

### Tech Stack
- React 18 | JavaScript | Axios | CSS
